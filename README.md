# All-In-One-Home-Automation-System-V1-
## All In One Home Automation System (V1)

##### Required Softwares 
- Arduino IDE (PC/ MAC/ LUNUX)
+ ESP RainMaker (Android/ IOS)
+ Optionals: Google Home, Amazon Alexa

##### Required libraries
 **ESP32** (boards package version:2.0.4) https://dl.espressif.com/dl/package_esp32_index.json
- DHT library(Version: 1.4.3) https://github.com/adafruit/DHT-sensor-library
- Sleep Timer library(Version: 1.0.0) https://github.com/kiryanenko/SimpleTimer
- Ace Button library (Version: 1.9.2) https://github.com/bxparks/AceButton


## Configuration
Download and instal Arduino IDE. Open Arduino IDE and go to files and then preferences, copy and paste the ESP32 board package version link (Provided in the required libraries section) in the additional boards manager url section, and download ESP32 board library version 2.0.4. Now download other three libraries, provided in th required libraries section. Now in Arduino Ide go to sketch tab and then click add file, now add the libraries one by one.
